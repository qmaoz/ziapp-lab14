# Top-words

Website for efficient and convenient German vocabulary learning

# Download the project

- Clone the repository:
```git clone https://github.com/qmaoz/top-words.git```

- Install all dependencies:
```npm install```

# Run the server

- Go to the "server" folder using a terminal:

```cd server```

> [!IMPORTANT]
> Run a DBMS on Your PC.

- Run the server:

```npm run start:dev```

# Run the client

- Go to the "client" folder using a terminal:

```cd client```

- Run the client:

```npm run start```

# Open the website

- Normally, the website opens automatically, but you can also open it manually using this URL:

```http://localhost:3000/```
